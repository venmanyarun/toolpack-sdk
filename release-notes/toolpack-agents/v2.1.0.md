# @toolpack-sdk/agents v2.1.0

Date: 2026-05-28

## Highlights

- **Released alongside `toolpack-sdk` v2.1.0** — version bumped to keep the package line aligned. No agent API changes.
- **Documentation now reflects stable status** — removed lingering alpha/pre-release indicators from the README.

---

## Breaking changes

None.

---

## Improvements

- **Peer dependencies bumped** — `toolpack-sdk` and `@toolpack-sdk/knowledge` peer ranges updated to `^2.1.0` (was `^2.0.0`) to match the lockstep release.
- **README** — removed the hardcoded `2.0.0-alpha.1` pre-release note, switched the npm badge from the `alpha` tag to the latest stable version, and dropped the `@alpha` tag from the install command. The package has been stable since `2.0.0`.
- **`SMSChannel`** — minor cleanup: removed an unused variable from the Twilio message-send path.

---

## Compatibility

- Node.js >= 20
- Peer dependency `toolpack-sdk` `^2.1.0`
- Peer dependency `@toolpack-sdk/knowledge` `^2.1.0`

## Install

```bash
npm install @toolpack-sdk/agents
```
