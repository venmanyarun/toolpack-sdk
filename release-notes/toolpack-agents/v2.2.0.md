# @toolpack-sdk/agents v2.2.0

Date: 2026-06-08

## Highlights

- **McpChannel** — expose any Toolpack agent as an MCP tool; pairs with `sdk.startMcpServer()` in `toolpack-sdk`

## New features

### `McpChannel`

Bridges a Toolpack agent into an MCP server. When an MCP client calls `tools/call` for `agent.<name>`, `McpChannel` routes the request to `agent.invokeAgent()` and returns the output as the MCP tool result.

```typescript
import { McpChannel } from '@toolpack-sdk/agents';
import { Toolpack } from 'toolpack-sdk';

const ch = new McpChannel();
const agent = new PrReviewerAgent({ channels: [ch] });
await agent.start();

const sdk = await Toolpack.init({ provider: 'anthropic', tools: true });
await sdk.startMcpServer({
  transport: 'stdio',
  agents: [ch.asAgentDefinition(agent)],
});
```

`ch.asAgentDefinition(agent)` produces an `McpAgentDefinition` with the agent's name, description, input schema, and an `invoke()` function that calls through to the agent. The definition is passed directly to `startMcpServer({ agents: [...] })`.

`isTriggerChannel = false` — the MCP client drives the conversation, so `ask()` works normally.

**Exports:** `McpChannel`, `McpChannelConfig`

### Eval primitives

A lightweight framework for testing agent answer quality and catching regressions.

```typescript
import {
  EvalDataset,
  EvalRunner,
  ContainsScorer,
  LLMJudgeScorer,
  CustomScorer,
  compareEvalRuns,
  formatEvalReport,
} from '@toolpack-sdk/agents';

const dataset = new EvalDataset([
  { id: 'q1', input: 'Capital of France?', expectedOutput: 'Paris' },
]);

const runner = new EvalRunner({ agent, dataset, scorers: [new ContainsScorer()] });
const run = await runner.run();
console.log(`Score: ${(run.averageScore * 100).toFixed(1)}%`);

// Regression detection
const report = compareEvalRuns(baselineRun, run);
console.log(formatEvalReport(report));
expect(report.regressions).toHaveLength(0); // CI gate
```

**Four built-in scorers:**

| Scorer | Description |
|---|---|
| `ExactMatchScorer` | Score 1.0 if output exactly matches expectedOutput (trimmed, case-insensitive by default) |
| `ContainsScorer` | Score 1.0 if output contains expectedOutput |
| `LLMJudgeScorer` | Asks an LLM to score the output on a 0–1 scale using a configurable prompt |
| `CustomScorer` | Supply any `score({ output, expectedOutput, case }) => Promise<number>` function |

**`EvalDataset`** holds `EvalCase[]` — each case has `id`, `input`, `expectedOutput`, and optional `metadata`. Supports `add()` and `filter()`.

**`EvalRunner`** supports `concurrency` for parallel case execution.

**`compareEvalRuns(baseline, current)`** returns an `EvalReport` with `regressions` and `improvements` arrays for each case where the score changed.

**`formatEvalReport(report)`** returns a human-readable table suitable for CI output.

**Exports:** `EvalDataset`, `EvalRunner`, `ExactMatchScorer`, `ContainsScorer`, `LLMJudgeScorer`, `CustomScorer`, `compareEvalRuns`, `formatEvalReport`, and all associated types (`EvalCase`, `EvalRun`, `EvalVerdict`, `EvalScoredResult`, `EvalScoredRun`, `EvalRegression`, `EvalImprovement`, `EvalReport`, `EvalRunnerOptions`, `EvalScorer`, `LLMJudgeScorerOptions`, `EvalCaseResult`)

### `createOTelTracerInterceptor`

An OpenTelemetry interceptor that wraps each agent invocation in a span. Compatible with any OTel-compliant backend (Jaeger, Honeycomb, Datadog, OTLP).

```typescript
import { createOTelTracerInterceptor } from '@toolpack-sdk/agents';

class MyAgent extends BaseAgent {
  interceptors = [
    createOTelTracerInterceptor({
      tracer: myTracerProvider.getTracer('my-agent'),
    }),
  ];
}
```

**Exports:** `createOTelTracerInterceptor`, `OTelSpanStatusCode`, `OTelTracerConfig`, `OTelTracerProvider`, `OTelTracer`, `OTelSpan`, `OTelSpanOptions`, `OTelSpanStatus`
