# @toolpack-sdk/agents v2.1.1

Date: 2026-05-28

## Highlights

- **Patch release** — corrects the internal peer dependency ranges that were left at `^2.0.0` in v2.1.0.

---

## Breaking changes

None.

---

## Fixes

- **Peer dependencies** — `toolpack-sdk` and `@toolpack-sdk/knowledge` peer ranges bumped to `^2.1.1` to match the lockstep release line. In v2.1.0 these were left at `^2.0.0`; installs still resolved correctly (the caret range includes 2.1.x), but the ranges now reflect the intended lockstep versions.

---

## Compatibility

- Node.js >= 20
- Peer dependency `toolpack-sdk` `^2.1.1`
- Peer dependency `@toolpack-sdk/knowledge` `^2.1.1`

## Install

```bash
npm install @toolpack-sdk/agents
```
